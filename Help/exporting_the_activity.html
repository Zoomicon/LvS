Exporting the activity

Once your activity is ready and you want to send it to others as a single file, it is recommended that you export it. LvS can pack activity folders into .lvs files.

Click on File > Export > Packed Activity.

You are asked to specify the name of the .lvs file as well as the folder where it will be created.

You can chose whether the .lvs file will include the multimedia file or not. Let's see the two different options in the following scenario:

The teacher prepares an activity, and exports it into a packed activity with multimedia which she sends to the student. The packed activity contains all the files the teacher had imported, including the multimedia file. The student works with the activity and exports it without multimedia in order to send it back to the teacher. The resulting packed activity contains all the activity elements except the multimedia file, which means that the size of the .lvs file is significantly reduced.


After you have exported the activity and close the application, you can start LvS by double-clicking directly on the .lvs file. LvS starts and a message appears asking you to specify the location where you want to unpack the activity folder.
