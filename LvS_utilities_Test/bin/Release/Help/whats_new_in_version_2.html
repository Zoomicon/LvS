What's new in version 2?

  1. Authoring mode (see Glossary in this tutorial)
  2. Importing (Packed activity, Video, Subtitles, Documents)
  3. Exporting (Packd activity, Subtitles, Documents)
  4. Student and Teacher general Notes and per subtitle Comments
  5. Multiple Documents in different tabs available
  6. Auto-resizing parts of the interface
  7. Option to write subtitle text directly under the player with simultaneous update of the subtitle grid.
  8. Most recently used files list
  9. Buttons "Set subtitle start" and "Set subtitle end" modifying the Start time and End time of an existing subtitle
