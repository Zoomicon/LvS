How do I open a ready-made activity?

In the LvS environment, an .lvs file is called packed activity.

An activity is a folder containing various elements (files), whereas a packed activity is a compressed activity folder which users can send to each other.

While you work with LvS, you open and close an activity (activity folder). When you receive and send your work, you import and export a packed activity.

If you have been sent an .lvs file (packed activity), you have to import it in order to see its contents and use it. When you import a packed activity, LvS unpacks it in a folder, in other words the .lvs file is de-compressed.

In order to import a packed activity follow these steps:


1. Click on File > Import > Packed activity > Browse...

2. Select the folder, select the file and click on the button "Open"


3. Select the parent folder where you want the activity to be unpacked (de-compressed).
Click the button "OK"


If a dialogue box appears with the message "Do you want to open or save this file"

1. Unselect the box "Always ask before opening this type of file"

2. Click the button "Open"
