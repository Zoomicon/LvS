Adding subtitles (cueing)

Make sure Authoring mode is enabled: Options > Authoring mode.

Watch the multimedia file and:

1. Click on the button "Pause" or press F5 at the point you want a new subtitle to appear.

2. Click on the button "Add subtitle". A new line will appear in the subtitle grid, just under the selected subtitle.


The Start time of this new line is as the time when when you clicked on "Add subtitle". The Duration is automatically set to 2 seconds, therefore the End time is equal to the Start time + 2 seconds.

Of course, since this duration is automatically set, it might not suit your needs. In order to modify the End time of the specific subtitle see section "Modifying subtitle's Start and End time".
