Shortcuts

F1 = Help

F2 = Save

F3 = Open activity

F4 = New activity

F5 = Play/Pause

F6 = Set subtitle start

F7 = Set subtitle end

F8 = Add subtitle

F9 = Remove subtitle

F10 = Open menu

F11 = Authoring mode (activate/deactivate)

F12 = Exit application or Alt+F4
